# Guide de Contribution

Merci de votre intérêt pour Perfect Models Management ! Nous sommes ravis que vous souhaitiez contribuer à notre projet.

## Comment contribuer

### Signaler un bug
1. Vérifiez que le bug n'a pas déjà été signalé dans les [issues](https://github.com/votre-utilisateur/perfect-models-management/issues)
2. Si le bug n'existe pas, créez une nouvelle issue avec une description claire
3. Incluez des étapes pour reproduire le bug
4. Ajoutez des captures d'écran si possible
5. Indiquez votre environnement (navigateur, OS, etc.)

### Proposer une amélioration
1. Vérifiez que l'amélioration n'a pas déjà été proposée
2. Créez une issue pour discuter de votre proposition
3. Une fois approuvée, vous pouvez commencer à travailler dessus

## Processus de développement

### Installation
Voir le guide d'installation dans [INSTALLATION.md](./INSTALLATION.md)

### Branches
- `main` - Branche de production
- `develop` - Branche de développement principale
- `feature/nom-de-la-fonctionnalité` - Pour les nouvelles fonctionnalités
- `bugfix/nom-du-correctif` - Pour les corrections de bugs

### Workflow Git
1. Créez une branche pour votre fonctionnalité
   ```bash
   git checkout -b feature/nouvelle-fonctionnalite
   ```
2. Faites vos modifications
3. Ajoutez vos changements
   ```bash
   git add .
   ```
4. Créez un commit descriptif
   ```bash
   git commit -m "Ajoute une nouvelle fonctionnalité"
   ```
5. Poussez vos changements
   ```bash
   git push origin feature/nouvelle-fonctionnalite
   ```
6. Créez une Pull Request vers `develop`

### Bonnes pratiques
- Écrivez du code propre et commenté
- Suivez les conventions de nommage
- Écrivez des tests pour votre code
- Mettez à jour la documentation si nécessaire
- Vérifiez que les tests passent avant de pousser

## Code de conduite

### Nos standards
- Soyez respectueux et bienveillant
- Acceptez les critiques constructives
- Montrez de l'empathie envers les autres contributeurs
- Soyez ouvert aux différentes perspectives et expériences

### Modération
Tout comportement inapproprié peut être signalé à contact@perfectmodels.ga

## Questions ?
N'hésitez pas à poser des questions en ouvrant une issue ou en nous contactant directement.
