# Guide d'Export PDF - Miss One Light 2026

## 📄 Format d'Export PDF

L'export génère un **document PDF professionnel** avec le classement des votes par candidate, sans informations financières.

## 📥 Comment exporter

### Depuis l'interface admin
1. Connectez-vous à `/admin-miss-one-light`
2. Allez dans l'onglet **"Votes en attente"** ou **"Comptabilité"**
3. Cliquez sur le bouton **"Exporter PDF"** 📄
4. Le fichier PDF est téléchargé automatiquement

## 📱 Utilisation du PDF

### Impression
1. Ouvrez le PDF avec Adobe Reader, Chrome, ou tout lecteur PDF
2. **Fichier** > **Imprimer**
3. Format : **A4**
4. Orientation : **Portrait**
5. Qualité : **Haute qualité** (300 dpi recommandé)

### Partage
- **Email** : Pièce jointe professionnelle
- **WhatsApp** : Envoi direct du document
- **Réseaux sociaux** : Partage du classement officiel
- **Impression** : Affichage lors de l'événement

### Archivage
- Conservez chaque export avec la date
- Format : `miss-one-light-classement-2026-04-17.pdf`
- Stockage cloud recommandé (Google Drive, Dropbox)

## 🆘 Problèmes courants

### Le PDF ne s'ouvre pas
- Vérifiez que vous avez un lecteur PDF installé
- Essayez avec Chrome, Firefox, ou Adobe Reader
- Téléchargez à nouveau si le fichier est corrompu

### Les couleurs ne s'affichent pas
- Vérifiez les paramètres de votre lecteur PDF
- Activez l'affichage des couleurs
- Mettez à jour votre lecteur PDF

### L'impression est de mauvaise qualité
- Vérifiez les paramètres d'impression (300 dpi minimum)
- Utilisez du papier de qualité
- Nettoyez les têtes d'impression

### Le fichier est trop lourd
- Le PDF est optimisé et pèse moins de 100 KB
- Si le problème persiste, vérifiez votre connexion
- Essayez de télécharger à nouveau
