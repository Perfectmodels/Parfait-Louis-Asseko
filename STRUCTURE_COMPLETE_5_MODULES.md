## 💡 Conseils pour la Génération de Contenu

### Structure de Chapitre Type
```typescript
{
  title: "Titre du Chapitre",
  content: [
    "Paragraphe 1 : Introduction et contexte",
    "Paragraphe 2 : Concepts clés",
    "Paragraphe 3 : Applications pratiques",
    "Paragraphe 4 : Exemples concrets",
    "Paragraphe 5 : Conseils et bonnes pratiques"
  ],
  keyPoints: [
    "Point clé 1",
    "Point clé 2",
    "Point clé 3",
    "Point clé 4",
    "Point clé 5"
  ],
  quiz: [
    // 20 questions avec structure :
    // - 5 questions de définition/compréhension
    // - 5 questions d'application pratique
    // - 5 questions de cas concrets
    // - 5 questions sur les erreurs à éviter
  ]
}
```

### Répartition des Questions (20 par chapitre)
1. **Questions 1-5** : Définitions et concepts clés
2. **Questions 6-10** : Applications pratiques
3. **Questions 11-15** : Cas concrets et situations réelles
4. **Questions 16-20** : Erreurs courantes à éviter

### Utilisation d'IA pour Générer les Questions
```
Prompt pour ChatGPT/Claude :

"Génère 20 questions QCM sur le thème '[TITRE DU CHAPITRE]' pour une formation de mannequins professionnels au Gabon.

Contexte : [COPIER LE CONTENU DU CHAPITRE]

Structure requise :
- Questions 1-5 : Définitions et concepts clés
- Questions 6-10 : Applications pratiques
- Questions 11-15 : Cas concrets et situations réelles
- Questions 16-20 : Erreurs courantes à éviter

Format pour chaque question :
{
  question: "...",
  options: ["Option A", "Option B", "Option C", "Option D"],
  correct: 0-3,
  explanation: "Explication détaillée de la bonne réponse"
}

Les questions doivent être :
- Pertinentes pour le contexte gabonais
- Pratiques et applicables
- Variées en difficulté
- Avec des explications pédagogiques"
```
