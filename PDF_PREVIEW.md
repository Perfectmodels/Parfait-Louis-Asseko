## 🎨 Légende des couleurs

### Dans le PDF réel :

**En-tête (fond doré)**
- Fond : #FCD116 (Jaune doré)
- Texte : #1a1a1a (Noir foncé)

**Résumé général**
- Labels : Gris foncé
- Valeurs : #FCD116 (Doré) - alignées à droite

**Tableau de classement**
- **En-tête** : Fond doré (#FCD116), texte noir
- **Pos.** : Centré, noir
- **Candidate** : Gras, noir
- **Validés** : Vert (#009E60), aligné à droite
- **En attente** : Orange (#FF9800), aligné à droite
- **Total** : Doré (#FCD116), gras, aligné à droite
- **Trans.** : Centré, noir
- **Lignes alternées** : Fond gris clair (#fafafa)

**Pied de page**
- Texte : Gris clair (#969696)
- Taille : 8pt
