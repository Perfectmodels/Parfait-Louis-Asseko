## 2026-03-16 - Accessibility of Mobile Toggles
**Learning:** Icon-only interactive elements like mobile navigation toggles require explicit ARIA attributes (`aria-label`, `aria-expanded`, `aria-controls`) and `focus-visible` styling (`focus-visible:ring-2`, `focus-visible:ring-pm-gold`) to adhere to accessibility and keyboard navigation standards in this application.
**Action:** Always ensure that icon-only buttons, especially those controlling interactive elements like overlays, are fully accessible with semantic ARIA labels and clear focus states.
