# Guide Rapide - Miss 5ème

## 🎯 Accès Rapide

### Pour les Jurés
**URL** : `votre-site.com/jury/miss-5eme`
- Choisir son numéro (Juré 1, 2, 3 ou 4)
- Entrer PIN : **0000**
- Commencer à noter !

### Pour l'Admin
**URL** : `votre-site.com/admin/miss-5eme`
- Menu : Admin > Recrutement > Miss 5ème

---

## 👥 Pour les Jurés

### Étapes de notation

1. **Se connecter**
   - Sélectionner votre numéro de juré (1-4)
   - Entrer le PIN 0000

2. **Choisir le passage**
   - Passage 1, 2 ou 3

3. **Sélectionner une candidate**
   - Cliquer sur le numéro de la candidate

4. **Noter les critères** (0-4 points chacun)
   - 😊 Sourire
   - 🤸 Gestuelle
   - ⭐ Performance Technique
   - 👗 Prestance et Élégance

5. **Valider**
   - Total max : 20 points
   - Enregistrer la note

### Indicateurs
- 🟢 **Vert** : Déjà noté
- 🔴 **Rose** : Sélectionné
- ⚪ **Gris** : À noter

---

## 🎨 Pour l'Administrateur

### Onglet 1 : Candidates
- ➕ Ajouter des candidates (max 10)
- 🔗 Copier le lien pour les jurés
- 🗑️ Supprimer une candidate
- 🔄 Réinitialiser toutes les notes

### Onglet 2 : Fiches Individuelles
- 📋 Voir les notes détaillées par candidate
- 👥 Notes de chaque juré
- 📊 Moyennes par passage
- 🎯 Note finale

### Onglet 3 : Classement Final
- 🏆 Classement automatique
- 🥇🥈🥉 Médailles pour le top 3
- 📈 Scores par passage
- 💯 Note finale sur 20

---

## 📊 Système de Notation

### Critères (4 points max chacun)
```
Sourire              : 0 - 4 points
Gestuelle            : 0 - 4 points
Performance Technique: 0 - 4 points
Prestance & Élégance : 0 - 4 points
─────────────────────────────────
TOTAL PAR PASSAGE    : 0 - 20 points
```

### Calcul Final
```
Note Passage 1 = Moyenne des 4 jurés
Note Passage 2 = Moyenne des 4 jurés
Note Passage 3 = Moyenne des 4 jurés

NOTE FINALE = (P1 + P2 + P3) / 3
```

---

## 🔐 Sécurité

✅ Un juré ne peut pas noter 2 fois la même candidate pour le même passage
✅ Le total ne peut pas dépasser 20 points
✅ Chaque juré a son numéro unique (1-4)
✅ Sauvegarde automatique en temps réel

## 💡 Astuces

### Pour les Jurés
- Notez au fur et à mesure, pas besoin d'attendre
- Les notes sont sauvegardées immédiatement
- Vous pouvez vous déconnecter et revenir plus tard
- Les candidates déjà notées sont marquées en vert

### Pour l'Admin
- Consultez les fiches individuelles pour voir les notes croisées
- Le classement se met à jour en temps réel
- Vous pouvez réinitialiser si besoin de recommencer
- Partagez le lien directement depuis l'interface

## 🆘 Problèmes Courants

**Q: Un juré a oublié son numéro**
R: Pas de problème, il peut choisir n'importe quel numéro libre (1-4)

**Q: Une note a été donnée par erreur**
R: L'admin peut réinitialiser toutes les notes ou supprimer la candidate

**Q: Le total dépasse 20**
R: Impossible, le système bloque l'enregistrement

**Q: Deux jurés veulent le même numéro**
R: Chaque numéro (1-4) ne peut être utilisé qu'une fois
